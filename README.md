# TaskFlow - Todo Dashboard

A modern and responsive Todo Dashboard application built with Next.js, Zustand, shadcn/ui, and Tailwind CSS. The project focuses on building a scalable task management system with clean UI, state management and reusable components.

## Tech Stack

- **Next.js** - React framework for building fast and scalable web applications
- **TypeScript** - Type-safe JavaScript development
- **Tailwind CSS** - Utility-first CSS framework for styling
- **shadcn/ui** - Reusable and accessible UI components
- **Zustand** - Lightweight global state management
- **React Icons** - Icon library for UI elements

## Features

### Task Management (CRUD)

- Create new tasks
- View all tasks
- Update/edit existing tasks
- Delete tasks
- Mark tasks as completed
- Manage task status

### Dashboard

- Responsive dashboard layout
- Collapsible sidebar navigation
- Task overview interface
- Analytics-ready structure

### State Management

- Global task state management using Zustand
- Centralized task operations
- Persistent task state handling

### UI & UX

- Modern UI using shadcn/ui components
- Responsive design with Tailwind CSS
- Clean and reusable components
- Accessible UI patterns
